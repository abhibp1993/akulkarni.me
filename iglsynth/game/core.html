Action¶

class iglsynth.game.core.Action(name=None, func=None)[source]¶

Bases: object

Represents an action. An action acts on a state (of TSys or Game etc.) to produce a new state.

Parameters:
  • name – (str) Name of the action.
  • func – (function) An implementation of action.

Note

Acceptable function templates are,

  • st <- func(st)
  • st <- func(st, *args)
  • st <- func(st, **kwargs)
  • st <- func(st, *args, **kwargs)
